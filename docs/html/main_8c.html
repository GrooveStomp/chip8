Detailed Description

This file is the main entrypoint for the CHIP-8 emulator. The program runs on four threads:

  • graphics + input
  • sound
  • timers
  • system emulation

The system emulation (re: "main") thread provides routines for other threads to call into, and these routines internally use POSIX threads synchronization primitives to sync data.

Besides creating the threads, this file is responsible for parsing CLI args, loading CHIP-8 ROM data and otherwise just being a main entrypoint.
