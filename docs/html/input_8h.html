Detailed Description

This package provides a small input abstraction layer to separate input processing from system emulation. Input operates in the same thread as graphics, at the same frequency (30hz) and calls back into the system emulation thread.

Input processing.

The main input processing routine. This interacts with SDL, contains the key mapping between SDL keys and the CHIP-8 hex keyboard, and calls synchronization routines on the system package to synchronize data across threads.
