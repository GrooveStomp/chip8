void* soundWork ( void *  context)

Thread for sound playback.

Sound playback is triggered via the CHIP-8's sound timer. When the timer reaches zero, a sound is played back. The specifications seem loose on what this means exactly, so this emulator plays back a tone of 440hz for 200ms.
