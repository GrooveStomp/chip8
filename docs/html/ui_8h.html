Detailed Description

The UI system uses nuklear internally for an immediate-mode UI. The UI isn't a general purpose UI, rather it's a graphical debugger embedded within the CHIP-8 emulator.

See also
Nuklear: https://github.com/vurtun/nuklear
Immediate mode GUI: https://caseymuratori.com/blog_0001
